# Netflixclone
Developed a replica of the official Netflix home page, featuring four distinct sections:

Header: Includes a navigation bar with language options and a "Sign up" button, alongside a compelling content message with a prominent Call-To-Action form.

Features: Comprises four segments, each showcasing a product image and its corresponding features to enhance user engagement.

FAQ: Utilizes an interactive design, where clicking the Plus button reveals answers to frequently asked questions, and the Close button conceals the responses. Additionally, a secondary Call-To-Action form is seamlessly integrated.

Footer: Incorporates a helpline number for immediate assistance, accompanied by multiple reference links for further exploration, contributing to a comprehensive and user-friendly browsing experience.

Language used: HTMl, CSS, JS 
Programming concepts: Event handling and Functions to control the visibility of element with button hover effects
